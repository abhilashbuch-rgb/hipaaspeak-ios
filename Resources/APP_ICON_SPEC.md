# App Icon Specification

## Design
- **Symbol:** Purple asterisk (*)
- **Color:** Brand purple — RGB(115, 51, 191) / Hex #7333BF
- **Background:** White or very light gray (#FAFAFA)
- **Style:** Bold, rounded weight, centered
- **Corner radius:** Applied automatically by iOS

## Required sizes
Create a single 1024x1024 PNG and Xcode will auto-generate all sizes.

Place in: `Resources/Assets.xcassets/AppIcon.appiconset/`

## In Xcode
1. Open Assets.xcassets
2. Click AppIcon
3. Drag the 1024x1024 PNG into the "All Sizes" slot
4. Xcode generates all required sizes automatically (iOS 17+)
